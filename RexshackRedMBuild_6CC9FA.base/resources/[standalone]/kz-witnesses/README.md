# RedM-Witnesses
A script that will transform an NPC into a witness when you shoot at other NPCs or players and notify law enforcement. 

**Dependencies**

- [rsg-core]
- [rsg-policejob]

**Instructions**

- Extract download into your resources folder
- Start the resource in your server.cfg
